Practice 11 - Pygame

Inside:
1. racer/main.py
2. snake/main.py
3. paint/main.py

How to run:

python3 racer/main.py
python3 snake/main.py
python3 paint/main.py

If pygame is not installed:

python3 -m pip install pygame

Racer:
- Left / Right arrows or A / D
- Coins have different weights
- Enemy speed increases after 5 coins

Snake:
- Arrow keys or W / A / S / D
- Food has different weights
- Food disappears after timer

Paint:
- Choose color from top panel
- Draw square, right triangle, equilateral triangle, rhombus
