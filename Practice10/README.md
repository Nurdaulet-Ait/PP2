Practice 10 - Pygame

This folder contains 3 completed tasks:

1. Racer
2. Snake
3. Paint

How to run

Open terminal in this folder and run:

python3 racer/main.py
python3 snake/main.py
python3 paint/main.py


If pygame is not installed:


python3 -m pip install pygame


Controls

Racer
- Left / Right arrows
- A / D

Snake
- Arrow keys
- W / A / S / D

Paint
- Choose color from top panel
- Choose Brush / Rect / Circle / Eraser
- Clear button clears canvas
